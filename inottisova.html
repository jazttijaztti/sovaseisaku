gitのリナックスコマンド

  • git init

    隠しファイル.gitがフォルダの中に生成される

  • ls -la

    フォルダ内の隠しファイルを表示

  • git branch

    自分で作成したbranchが表示され、現在選択中のbranchがわかる

    デフォルトでmasterやmainといったbranchがデフォルトで入っている

  • git branch x

    "x"という名前のbranchを作成する

  • git add .

    現在の作業中のフォルダ全体を選択する

  • git commit -am "x"

    git add .で現在作業中のフォルダの変更差分を現在選択中のbranchに同期する

    同期の際に、"x"というコメントを残す

  • git checkout x

    "x"という名前のbranchに切り替える

  • git merge x

    "x"という名前のbranchの変更差分を現在選択中のbranchに取り込む

  • git push origin x

    現在選択中branchの変更差分を共同作業のフォルダに同期する

  • git pull origin x

    共同作業を行なっているる別の人がpushした内容を現在自分の作業しているbrnchに同期する
